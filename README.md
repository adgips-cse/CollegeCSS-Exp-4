# CollegeCSS-Exp-4
Experiment-4 : Use External, Internal, and Inline CSS to format the college web page that you created.

Aim :- To design and format a college web page using Inline CSS, Internal CSS, and External CSS.

Software Required
VS Code / Notepad
Web Browser (Chrome / Edge)

Theory
Inline CSS is applied directly inside an HTML tag using the style attribute.
Internal CSS is written inside the <style> tag within the <head> section.
External CSS is written in a separate .css file and linked to the HTML file using the <link> tag.

Procedure
Create a folder named CollegeCSS.
Create two files: index.html and style.css.
Apply External CSS using style.css.
Apply Internal CSS inside the <style> tag.
Apply Inline CSS using the style attribute.
Open index.html in a web browser to view the formatted webpage.
